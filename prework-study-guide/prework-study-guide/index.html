CSS

  • Declarations: "color: blue;" CSS > color, Value > blue.
  • Margin: how much space we want around the outside of an element.
  • Padding: the amount of space we want around the content on the inside of an element.
  • google CSS box model.
  • *: Apply to all of the elements on a page
  • DRY: do not repeat yourself, put two things that can be styled together in the same group.
